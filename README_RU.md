# ComfyUI-Krea2-Projector-Tuner (BSS)

🇷🇺 Русский | 🇺🇸 [Read in English](README.md)

**ComfyUI-Krea2-Projector-Tuner (BSS)** — это специализированный кастомный нод для ComfyUI, позволяющий на лету в оперативной памяти модифицировать и настраивать слои `txtfusion.projector` моделей **Krea2**. Нод предоставляет ползунки управления в реальном времени и динамические пресеты (включая лучшую конфигурацию **Strongx3_wl7**) для обхода фильтров безопасности, регулировки стиля и контроля следования промпту без сохранения временных файлов на диске.

## ⚡ Ключевые возможности (v2.0.0)

* **Динамический патчинг в памяти:** Использует встроенную систему наложения патчей весов ComfyUI. Изменения вносятся временно в оперативную память непосредственно во время генерации, что гарантирует 100% безопасность и исключает риск повреждения исходных файлов модели.
* **Динамическая папка пресетов на JSON:** Дефолтные пресеты автоматически создаются на первом запуске в виде отдельных файлов `.json` в папке `presets/` (`strongx3_wl7.json`, `strong_2.5.json`, `clean_bypass.json`). Вы можете настраивать их, переименовывать, делиться ими с сообществом или удалять прямо в проводнике.
* **Удобные интерактивные кнопки на ноде:** Сохраняйте, удаляйте и открывайте папку с вашими конфигурациями весов прямо из интерфейса нода (UI обновляется мгновенно без перезагрузок страницы).
* **Автоматическая синхронизация веб-интерфейса:** Значения на слайдерах холста обновляются автоматически при выборе пресета. Изменение любого ползунка вручную мгновенно возвращает выпадающий список в положение `custom`.
* **Привычный сотый шаг (`0.01`):** Комфортное перетаскивание ползунков с шагом `0.01` (как в стандартном LoRA Loader) с полным сохранением возможности ввода высокоточных дробных значений с клавиатуры (до 6 знаков после запятой).

## 📥 Установка

### Способ 1: Через ComfyUI Manager (Рекомендуется)
1. Откройте ComfyUI и нажмите кнопку **Manager**.
2. Нажмите **Install Custom Nodes**.
3. Введите в поиске **`Krea2 Projector Tuner`**.
4. Нажмите **Install**, дождитесь завершения установки и перезагрузите ComfyUI.

### Способ 2: Через Git URL (Если нод еще не проиндексирован)
1. Откройте ComfyUI Manager -> нажмите **Install via Git URL**.
2. Вставьте ссылку на этот репозиторий: `https://github.com/BlackSnowSkill/ComfyUI-Krea2-Projector-Tuner`
3. Нажмите **Install** и перезагрузите ComfyUI.

### Способ 3: Ручная установка (Git Clone)
1. Откройте терминал в папке кастомных нод: `ComfyUI/custom_nodes`.
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/BlackSnowSkill/ComfyUI-Krea2-Projector-Tuner.git
   ```
3. Перезапустите ComfyUI.

## 🧩 Описание нода

### 🎛️ Krea2 Projector Tuner (BSS) (класс `Krea2ProjectorTuner`)
Зарегистрирован в категории `BSS/Krea2`.

* **Входные параметры:**
  * **`model`:** Загруженная модель Krea2.
  * **`preset`:** Выбор пресета из файлов JSON в папке `presets/`, или `custom` (для активации ручных ползунков).
  * **`multiplier`:** Общий множитель весов (от -10.0 до +10.0) с шагом `0.01` (по умолчанию `1.0`). Умножает значения всех выбранных ручек, позволяя легко масштабировать силу эффекта (например, в 3x или 5x).
  * **`knob_0` – `knob_11`:** Множители смещения весов (от -5.0 до +5.0) с шагом `0.01`.

---

## 🎛️ Управляющие кнопки на ноде

* **`Save Preset`:** Выводит окно для ввода имени и сохраняет текущие ручные значения ползунков как новый файл JSON в папке `presets/`. Список пресетов обновляется мгновенно на экране.
* **`Delete Selected Preset`:** Удаляет файл выбранного пресета с диска и возвращает нод в режим `custom`.
* **`Open Presets Folder`:** Мгновенно открывает папку с пресетами в проводнике Windows.

## 📋 Как работают ползунки (слои)

12 регуляторов соответствуют скрытым состояниям текстового энкодера CLIP и отвечают за разные уровни абстракции информации:
* **Слайдеры 0–1 (Ранние слои):** Отвечают за базовый синтаксис, распознавание слов и прямое следование промпту.
* **Слайдеры 2–6 (Средние слои):** Управляют геометрией, объектами, композицией и пропорциями человека (выступают в роли стабилизатора анатомии).
* **Слайдеры 7–10 (Глубокие слои):** Отвечают за абстрактные темы, стили и негативное выравнивание (безопасность/цензура). Смещение их в отрицательную область обходит цензуру.
* **Слайдер 11 (Финальный слой):** Финальный выходной слой текстового энкодера CLIP.
