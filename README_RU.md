# ComfyUI-Krea2-Projector-Tuner (BSS)

🇷🇺 Русский | 🇺🇸 [Read in English](README.md)

**ComfyUI-Krea2-Projector-Tuner (BSS)** — это специализированный кастомный нод для ComfyUI, позволяющий на лету в оперативной памяти модифицировать и настраивать слои `txtfusion.projector` моделей **Krea2**. Нод предоставляет ползунки управления в реальном времени и оптимизированные пресеты (включая лучшую конфигурацию **Strongx3_wl7**) для обхода фильтров безопасности, улучшения анатомии и контроля следования промпту без сохранения временных файлов на диске.

## ⚡ Ключевые возможности

* **Динамический патчинг в памяти:** Использует встроенную систему наложения патчей весов ComfyUI. Изменения вносятся временно в оперативную память непосредственно во время генерации, что гарантирует 100% безопасность и исключает риск повреждения исходных файлов модели.
* **Встроенные премиум-пресеты:**
  * **`Strongx3_wl7` (По умолчанию):** Самый совершенный патч обхода фильтра. Применяет тройное (3.0x) подавление слоев безопасности 8–10 и точечное смещение `-0.7` на 7-м слое для блокировки последнего пика безопасности, обеспечивая максимально чистый обход цензуры.
  * **`Strong 2.5`:** Классический популярный патч обхода фильтра.
  * **`Anatomy Opt`:** Создан для исправления анатомических дефектов за счет буста средних слоев (пропорции/геометрия) при сохранении подавления фильтра цензуры.
  * **`Clean Bypass`:** Стандартный чистый обход фильтра.
* **12 ползунков управления (`knob_0` – `knob_11`):** Прямое сопоставление с 12 слоями текстового энкодера CLIP, выступающими в роли индивидуальных семантических слайдеров (стиль, детализация, геометрия и цензура).

## 📥 Установка

### Способ 1: Через ComfyUI Manager (Рекомендуется)
1. Откройте ComfyUI и нажмите кнопку **Manager**.
2. Нажмите **Install via Git URL**.
3. Вставьте ссылку на этот репозиторий: `https://github.com/BlackSnowSkill/ComfyUI-Krea2-Projector-Tuner`
4. Нажмите **Install**, дождитесь завершения установки и перезагрузите ComfyUI.

### Способ 2: Ручная установка (Git Clone)
1. Откройте терминал и перейдите в папку кастомных нод ComfyUI:
   ```bash
   cd ComfyUI/custom_nodes
   ```
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/BlackSnowSkill/ComfyUI-Krea2-Projector-Tuner.git
   ```
3. Перезапустите ComfyUI.

## 🧩 Описание нода

### 🎛️ Krea2 Projector Tuner (BSS) (класс `Krea2ProjectorTuner`)
Зарегистрирован в категории `BSS/Krea2`.

* **Входные параметры:**
  * **`model`:** Загруженная модель Krea2.
  * **`preset`:** Выбор пресета (`Strongx3_wl7`, `Strong 2.5`, `Anatomy Opt`, `Clean Bypass` или `custom` для активации ручных ползунков).
  * **`knob_0` – `knob_11`:** Множители смещения весов (от -5.0 до +5.0) для каждого из 12 слоев проектора CLIP.

---

## 📋 Как работают ползунки (слои)

12 регуляторов соответствуют скрытым состояниям текстового энкодера CLIP и отвечают за разные уровни абстракции информации:
* **Слайдеры 0–1 (Ранние слои):** Отвечают за базовый синтаксис, распознавание слов и прямое следование промпту.
* **Слайдеры 2–6 (Средние слои):** Управляют геометрией, объектами, композицией и пропорциями человека. Их буст выступает в роли **слайдера анатомии**, исправляя дефекты конечностей.
* **Слайдеры 7–10 (Глубокие слои):** Отвечают за абстрактные темы, стили и негативное выравнивание (безопасность/цензура). Смещение их в отрицательную область обходит цензуру.
* **Слайдер 11 (Финальный слой):** Финальный выходной слой текстового энкодера CLIP.
